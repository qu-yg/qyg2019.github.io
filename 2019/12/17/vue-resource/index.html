vue-resource的使用

Vue 要实现异步加载需要使用到 vue-resource 库。

Vue.js 2.0 版本推荐使用 axios 来完成 ajax 请求。

vue-resource 的配置步骤：

  • 直接在页面中，通过script标签，引入 vue-resource 的脚本文件；
  • 注意：引用的先后顺序是：先引用 Vue 的脚本文件，再引用 vue-resource 的脚本文件；
